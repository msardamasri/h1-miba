# HACKATHON - MIBA

Hackathon project: EDA and advanced feature engineering on Airbnb dataset

## Installation

1. Clone the repository:

```bash
git clone https://github.com/msardamasri/h1-miba.git
cd h1-miba
```

2. Create the environment (with micromamba):

```bash
micromamba create h1-miba -f environment.yml -y
micromamba activate h1-miba
```

3. Install the package in editable mode:

```bash
pip install -e .
```

## Running Notebook

To run all the notebook execute the main one in the package.
